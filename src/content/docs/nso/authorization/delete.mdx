<Path method="DELETE" path="/authorization/delete" base={Routes.nso} />

Delete the database entry for the given session token

This does not invalidate the session token! You have to manually revoke it from the [login history](https://accounts.nintendo.com/login_history) on the Nintendo Account website.

</AuthorizationHandler>
